Shadow Mode

ExtagDOM预定义了shadow-mode特性，如果shadow-mode="open"(不支持其他模式)，ExtagDOM会将子节点渲染到元素的shadowRoot中。

<div>
  <div>
    <span>123</span>
  </div>
  <div shadow-mode="open">
    <span>456</span>
  </div>
</div>

在 Web Components 中使用 Extag

通过attach方法，可以将Extag.Component与Web Components结合起来。

class Hello extends Extag.Component {
  static get template() {
    return `
      <x-hello shadow-mode="open">
        <h2>Hello, @{name}!</h2>
      </x-hello>
    `
  }
  static get attributes() {
    return {
      name: 'World'
    }
  }
}
​
class XHello extends HTMLElement {
  static get observedAttributes() {
    return ['name'];
  }
  connectedCallback() {
    this.hello = new Hello();
    this.hello.attach(this);
    for (let attr of this.attributes) {
      this.hello.set(attr.name, attr.value);
    }
  }
  attributeChangedCallback(name, oldValue, newValue) {
    if (this.hello && oldValue !== newValue) {
      this.hello.set(name, newValue);
    }
  }
}
​
customElements.define('x-hello', XHello);
